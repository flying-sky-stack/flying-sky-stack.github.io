    • 最近刷B站又被一个"老朋友"刷屏了——HwLOS，这个号称"自主研发"的操作系统，在2025年9月被扒皮后，不仅没凉，反而更新了10个版本，还建了自己的官网。今天必须来扒一扒这场荒诞的"字研"闹剧。
      • \n
    • 二、最大破绽：安装程序呢？
      \n最离谱的是，这系统根本没有安装过程。
      \n正常操作系统：插入U盘 → 分区 → 格式化 → 复制文件 → 安装驱动 → 重启配置 → 进入桌面（耗时20-40分钟）
      \nHwLOS：插入U盘 → 直接进桌面。
      \n还没开始安装呢，就已经"开机"了。这不就是典型的PE系统（预安装环境）特征吗？
      \n作者可能觉得把Windows安装界面删掉，用户就看不出来这是PE套壳了。但稍微有点电脑常识的人都知道：真正的操作系统是需要安装的，不是插U盘直接进的。
      \n这就好比有人卖你"自主研发的新车"，结果你一坐进去发现是辆出租车，计价器还在跳表——连伪装都懒得做全套。
      • \n
    • 三、技术"创新"清单
      \n让我们看看这位"系统开发者"的核心技术栈：
      \n号称功能实际实现自研文件管理器7-Zip.exe 重命名8K超清壁纸实际不到1080P系统优化删除Windows组件免安装启动PE环境直接运行磁盘管理工具集成DiskGenius办公软件集成LibreOffice
      \n最讽刺的是，当我用鲁大师检测时，显示结果赫然是："Windows 11 精简版"。
      \n!
      \n估计鲁大师也没想到，自己有生之年还能"适配"所谓的"自研操作系统"。
