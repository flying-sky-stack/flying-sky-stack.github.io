DN002：HwLOS"字研系统"：一场关于"自主研发"的荒诞喜剧

最近刷B站又被一个"老朋友"刷屏了——HwLOS，这个号称"自主研发"的操作系统，在2025年9月被扒皮后，不仅没凉，反而更新了10个版本，还建了自己的官网。今天必须来扒一扒这场荒诞的"字研"闹剧。

一、事件回顾：从"自研系统"到"套壳现形"
2025年9月7日，B站UP主 HardwareLab 发布视频，高调宣布："自研操作系统！不是Windows魔改！"
结果不到24小时，技术博主拆包发现：

  • 系统内赫然存在
    Microsoft 和 Windows 文件夹
  • 内核就是
    Windows 11 的精简版
  • 所谓的"文件资源管理器"直接用的
    7-Zip（没错，就是那个压缩软件）

    这操作就好比把别人的车换个车标，然后说"这是我自主研发的汽车"。

二、最大破绽：安装程序呢？
最离谱的是，这系统根本没有安装过程。
正常操作系统：插入U盘 → 分区 → 格式化 → 复制文件 → 安装驱动 → 重启配置 → 进入桌面（耗时20-40分钟）
HwLOS：插入U盘 → 直接进桌面。
还没开始安装呢，就已经"开机"了。这不就是典型的PE系统（预安装环境）特征吗？
作者可能觉得把Windows安装界面删掉，用户就看不出来这是PE套壳了。但稍微有点电脑常识的人都知道：真正的操作系统是需要安装的，不是插U盘直接进的。
这就好比有人卖你"自主研发的新车"，结果你一坐进去发现是辆出租车，计价器还在跳表——连伪装都懒得做全套。

三、技术"创新"清单
让我们看看这位"系统开发者"的核心技术栈：
号称功能实际实现自研文件管理器7-Zip.exe 重命名8K超清壁纸实际不到1080P系统优化删除Windows组件免安装启动PE环境直接运行磁盘管理工具集成DiskGenius办公软件集成LibreOffice
最讽刺的是，当我用鲁大师检测时，显示结果赫然是："Windows 11 精简版"。

估计鲁大师也没想到，自己有生之年还能"适配"所谓的"自研操作系统"。

四、荒诞后续：越骂越更新
按理说，被扒成这样应该删号跑路了吧？
不，他更新了10个版本。
从1.0.1到6.5，更新日志写得有模有样：

  • "去除Press any key to boot提示"
  • "修改X盘卷标"
  • "应用8K壁纸"
  • "更新DiskGenius为6.0专业版"

    甚至还建了官网（hwlos.mysxl.cn），提供123云盘下载。
    这心理素质，这产品迭代速度，某些大厂PM看了都自愧不如。

五、为什么大家如此愤怒
说实话，如果只是做个Windows PE工具，或者明确说"基于Windows的定制系统"，没人会喷。
愤怒的核心在于欺骗：

  • 把PE环境包装成"操作系统"
  • 把免安装启动说成"即开即用优化"
  • 把删除组件说成"深度优化"
  • 把换壁纸称为"8K视觉体验"
  • 被实锤后仍坚称"不是Windows魔改"
    这消耗的是真正国产操作系统开发者的公信力。

六、总结
这场闹剧最讽刺的莫过于：如果作者当初诚实地说这是"基于Windows PE的维护工具"，或者真的用Linux内核套壳，都不至于被群嘲成这样。
"字研操作系统"（自主研发）这个谐音梗，或许就是技术社区给这类行为最精准的注脚。
最后送HwLOS作者一句话：
删组件不是研发，换壁纸不是创新，7-Zip不是文件管理器，PE环境不是操作系统，插U盘直接进桌面更不是"免安装黑科技"。
